# 🧠 Student Mental Health Risk Prediction

## 📌 Project Overview

This project uses machine learning to predict whether a student is likely to report depression based on demographic and academic information.

The project demonstrates the complete machine learning workflow, including data cleaning, exploratory data analysis (EDA), feature engineering, model training, evaluation, and deployment using Streamlit.

---

## 🎯 Research Question

Can demographic and academic characteristics be used to predict whether a college student reports depression?

---

## 📂 Dataset

- Student Mental Health Dataset
- 101 student survey responses
- Features include:
  - Age
  - Gender
  - Academic Field
  - Year of Study
  - CGPA
  - Marital Status

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

---

## 🤖 Machine Learning Models

This project compares two machine learning models:

- Logistic Regression
- Random Forest

The models were evaluated using:

- Accuracy
- Confusion Matrix
- Precision
- Recall
- F1-score
- Feature Importance

---

## 📈 Results

- Logistic Regression Accuracy: **71.43%**
- Random Forest Accuracy: **66.67%**

The Logistic Regression model achieved the highest accuracy on the original dataset.

## 💻 Running the Project


